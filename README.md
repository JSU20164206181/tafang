# tafang
简单的塔防游戏  
使用Swing 界面开发的一个塔防小游戏，初始生成一个游戏界面，通过控制鼠标来放置塔，自动生成怪物，并向着目的地移动，放塔消耗金币，杀死怪物获取金币和积分，根据积分排名
